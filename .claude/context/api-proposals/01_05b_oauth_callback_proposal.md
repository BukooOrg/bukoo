## Endpoint

| Field  | Value                                        |
| ------ | -------------------------------------------- |
| Method | GET                                          |
| URL    | `/api/app/v1/auth/oauth/{provider}/callback` |
| Auth   | None                                         |

This URL is registered with the OAuth provider (e.g., Google Cloud Console) as the authorized redirect URI. Google redirects the browser here after the user grants or denies consent. This endpoint never returns a JSON body — it always responds with a `302` redirect.

---

## Request

### Headers

_None required — this request originates from a browser redirect, not the frontend app._

### Path Parameters

| Parameter | Type   | Description                         |
| --------- | ------ | ----------------------------------- |
| provider  | string | OAuth provider name (e.g. `google`) |

### Query Parameters

| Parameter | Type   | Required | Description                                                           |
| --------- | ------ | -------- | --------------------------------------------------------------------- |
| code      | string | Yes\*    | Authorization code issued by the provider after user consent          |
| state     | string | Yes\*    | CSRF state token originally generated by `1.5a`                       |
| error     | string | No       | Set by the provider if the user denied consent (e.g. `access_denied`) |

\* `code` and `state` are absent when the provider sends an `error` param (user denied consent).

### Request Body

_None_

---

## Response

This endpoint **always** responds with a `302 Redirect`. It never returns a JSON body.

### Success Response

**Status:** 302 Found  
**Location:** `{FRONTEND_URL}/oauth/callback`  
**Cookie set:** `access_token` (HTTP-only, same-site, matches existing `set_auth_cookie()` behaviour)

The frontend `/oauth/callback` page reads the cookie and proceeds to the authenticated session.

### Error Response

**Status:** 302 Found  
**Location:** `{FRONTEND_URL}/oauth/callback?error={error_code}`

| Scenario                                  | `?error=` value            |
| ----------------------------------------- | -------------------------- |
| Provider not in registry                  | `OAUTH_PROVIDER_NOT_FOUND` |
| `code` or `state` missing from query      | `OAUTH_STATE_INVALID`      |
| Provider sent `error` param (user denied) | `OAUTH_STATE_INVALID`      |
| State not found in Redis (expired/reused) | `OAUTH_STATE_INVALID`      |
| Google token exchange or userinfo failed  | `GOOGLE_OAUTH_ERROR`       |
| User account is suspended                 | `USER_SUSPENDED`           |

The frontend `/oauth/callback` page reads `?error=` and renders the appropriate error message. **No raw JSON is ever returned** because this endpoint is the target of a browser redirect and displaying JSON to the user would be a broken UX.

---

## Procedures

The route handler wraps the entire flow in a `try/except DomainException` block. All domain exceptions are caught and converted to redirect responses — they are never allowed to reach the global `domain_exception_handler`.

**Step-by-step:**

1. If the query string contains an `error` param (e.g. `error=access_denied`), the provider reported a consent failure. Return immediately: redirect to `{FRONTEND_URL}/oauth/callback?error=OAUTH_STATE_INVALID`.

2. If `code` or `state` are absent from the query string, raise `OAuthStateInvalidError`.

3. Look up `{provider}` in the `OAuthProviderRegistry`. If absent, raise `OAuthProviderNotFoundError`.

4. Retrieve the `IOAuthProviderFactory` for the provider. Call `factory.create_provider()` — **factory method invocation** — returning the concrete `IOAuthProvider` (e.g., `GoogleOAuthProvider`). Pass this instance and remaining dependencies to `OAuthCallbackUseCase`.

5. `OAuthCallbackUseCase.execute(OAuthCallbackCommand(code=code, state=state))`:

   a. **Verify CSRF state:** call `cache_svc.exists(f"oauth_state:{state}")`. If `False` (expired after 10 min or already used), raise `OAuthStateInvalidError`. Immediately delete the state: `cache_svc.delete(f"oauth_state:{state}")` — state tokens are single-use.

   b. **Exchange code for token:** call `await self._oauth_provider.get_access_token(code)`. `GoogleOAuthProvider.get_access_token()` POSTs to `https://oauth2.googleapis.com/token`. If `httpx.HTTPStatusError` is raised (bad or already-used code), catch it and raise `GoogleOAuthError`.

   c. **Fetch user identity:** call `await self._oauth_provider.get_user_info(access_token)` → `OAuthUserInfo(id, email, name, avatar_url)`. `GoogleOAuthProvider.get_user_info()` GETs `https://www.googleapis.com/oauth2/v3/userinfo` (OpenID Connect v3; user ID is the `sub` field). If the request fails, raise `GoogleOAuthError`.

   d. **Find or create user** — call `account_repo.find_by_provider_and_open_id(AuthProvider.GOOGLE, user_info.id)`.

   **Branch A — Existing account link found:**
   - Load `UserEntity` via `user_repo.find_by_id(existing_account.user_id)`. If the record is missing (data-integrity edge case), fall through to Branch B.
   - If `user.status == UserStatus.SUSPENDED`, raise `UserSuspendedError`.
   - Refresh the stored OAuth token: `existing_account.update_token(access_token)`, then `account_repo.save(existing_account)`.
   - Stamp login time: `user.record_login()`, `user_repo.save(user)`.

   **Branch B — No account link found:**
   - `user_repo.find_by_email(user_info.email)`.
   - If a user exists by email:
     - If `status == UserStatus.SUSPENDED`, raise `UserSuspendedError`.
     - If `status == UserStatus.PENDING`, call `user.activate()` — Google's verified email claim makes this safe.
   - If no user exists: create a new `UserEntity` with `_id=str(uuid7())`, `_status=UserStatus.ACTIVE`, `_role=UserRole.USER`, `_hashed_password=None`, `_avatar_url=user_info.avatar_url`, `_date_of_birth=datetime.now(UTC).date()` (placeholder — not provided by Google). Call `user_repo.save(user)`.
   - Create a new `AccountEntity` with `_id=str(uuid7())`, `_user_id=user.id`, `_provider=AuthProvider.GOOGLE`, `_open_id=user_info.id`, `_encrypted_token=access_token`. Call `account_repo.save(account)`.
   - `user.record_login()`, `user_repo.save(user)`.

   e. **Commit:** `await self._db_session.commit()` — single commit covers all mutations above.

   f. **Issue token:** `token = self._token_svc.create_access_token(user.id)`. Return `TokenDTO(access_token=token)`.

6. Back in the route handler: create `redirect = RedirectResponse(url=f"{configs.FRONTEND_URL}/oauth/callback", status_code=302)`. Call `set_auth_cookie(redirect, result.access_token)`. Return the redirect.

7. **Exception path (any `DomainException` raised in steps 1–5):** look up the exception type in `EXCEPTION_MAP` to get the `ErrorCode` string. Return `RedirectResponse(url=f"{configs.FRONTEND_URL}/oauth/callback?error={error_code}", status_code=302)`.

## Factory Method Pattern Application

Same `IOAuthProvider` / `IOAuthProviderFactory` hierarchy introduced in `01_05a`.  
`OAuthCallbackUseCase` receives `IOAuthProviderFactory` — it calls `factory.create_provider()` and uses the returned `IOAuthProvider` for `get_access_token()` and `get_user_info()`. The use case has no knowledge of `GoogleOAuthProvider` specifically.

### Route handler error-handling pattern (reference)

```python
@router.get("/oauth/{provider}/callback", include_in_schema=False)
async def oauth_callback(
    provider: str,
    response: Response,
    registry: OAuthProviderRegistry,
    db_session: DbSession,
    cache_svc: CacheService,
    user_repo: UserRepo,
    account_repo: AccountRepo,
    token_svc: TokenService,
    code: str | None = None,
    state: str | None = None,
    error: str | None = None,
) -> RedirectResponse:
    configs = get_configs()
    frontend_callback = f"{configs.FRONTEND_URL}/oauth/callback"

    try:
        if error or not code or not state:
            raise OAuthStateInvalidError()

        factory = registry.get(provider)
        if not factory:
            raise OAuthProviderNotFoundError(provider)

        use_case = OAuthCallbackUseCase(
            db_session=db_session,
            cache_svc=cache_svc,
            user_repo=user_repo,
            account_repo=account_repo,
            factory=factory,
            token_svc=token_svc,
        )
        result = await use_case.execute(OAuthCallbackCommand(code=code, state=state))

        redirect = RedirectResponse(url=frontend_callback, status_code=302)
        set_auth_cookie(redirect, result.access_token)
        return redirect

    except DomainException as exc:
        mapping = EXCEPTION_MAP.get(type(exc))
        error_code = mapping.code if mapping else ErrorCode.INTERNAL_ERROR
        return RedirectResponse(
            url=f"{frontend_callback}?error={error_code}", status_code=302
        )
```
